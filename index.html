Patiencemeld

Meld diff tool using patience algorithm

Download this project as a .zip file Download this project as a tar.gz file

Could be meld better? It could.

As a developer I found meld, a very simplistic, handy and powerful comparator. Through years of code development I found myself connecting it to many (and I said many many many) other programs and script to acomplish my job.

So stop. go back to the start. Do not connect a zenity dialog with a script in bash to open a meld diff beetwen two working copies of two branches from a svn repo because you cant find the right graphic tool, just do the tool!

There I was. With a basic skills on python and gtk, but with a proof of concept who works, meld can do it right now, but It can do it better with some help.

Learn more about PatienceMeld

I choose the name in a five minutes reflexion, comparing the objetives of Patience algorithm with mine. Do the same, just waste some time to do it better.
